# Soil-Type-Classification-Using-Deep-Learning-techniques-using-LANDSAT-images

## Overview

This project focuses on classifying soil types using deep learning techniques applied to LANDSAT satellite imagery. The model is designed to analyze multispectral images and predict different soil categories.

## Objective

To develop a Convolutional Neural Network (CNN) model that accurately classifies soil types from satellite images using deep learning approaches.

## Dataset

* LANDSAT satellite imagery
* Multispectral image data used for training and testing

## Methodology

* Image preprocessing and normalization
* Handling noise and class imbalance
* Feature extraction using CNN
* Model training and evaluation

## 🛠️ Tech Stack

* Python
* TensorFlow / Keras
* OpenCV
* NumPy
* Matplotlib

## Model

* CNN-based deep learning architecture
* Trained on LANDSAT multispectral images
* Evaluated using accuracy and loss metrics

## Results

* Achieved effective classification performance on satellite imagery dataset
* Improved accuracy through preprocessing and class balancing techniques

## Publication

Published at IACIT-2023: *High-Resolution Satellite Image Classification using Deep Learning Techniques*
